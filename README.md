|---|---|

Integrated **Gemini API** to analyze task parameters such as deadline, difficulty, importance, and estimated hours. The app generates AI priority scores, smart reminders, next-action suggestions, and personalized study timelines.

Delivered **20+ features**, including task CRUD, subtask checklists, priority matrix, progress analytics dashboard, achievement badges, category filtering, dark/light mode toggle, and fully responsive UI.
